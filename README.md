# Автоматизированное UI-тестирование

Проект содержит автоматизированные end-to-end (e2e) тесты для проверки веб-интерфейсов (веб-форма, калькулятор, интернет-магазин). Тесты написаны на Python с использованием фреймворка Pytest, библиотеки Selenium WebDriver и Allure для формирования детализированных отчетов.

## Запуск тестов и формирование отчета

Чтобы запустить тесты и собрать данные для Allure-отчета, выполните команду в корневой директории проекта:
```bash
pytest --alluredir=allure-results
```

## Просмотр сформированного отчета

После завершения прогона тестов папка allure-results наполнится файлами с результатами. Чтобы сгенерировать из них читабельный HTML-отчет и сразу открыть его в браузере, выполните команду:

```bash
allure serve allure-results
```

Для остановки локального сервера нажмите Ctrl+C в терминале.
